First Project: Scratch

This is our scratch game. The goal was to create a game where the goal is to move the avatar to hit the ball to touch the grey button. As you continue the game and score more points, the button will eventually start to move, making the game harder and create different levels. In order to do this, our code has some logic to where the code would read the global score and compare it to a set number. Then using an "If" code, the program would make the button move if the score makes the statement true.
